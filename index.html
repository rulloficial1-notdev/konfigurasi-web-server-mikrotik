Konfigurasi Lengkap Routing, DHCP, NAT & Web Server Tanpa Internet

Lihat Tutorial Lengkap

Konfigurasi Full Step by Step (Winbox 64x)

Reset Router

System → Reset Configuration → Centang No Default Configuration.

Set IP Address

IP → Addresses → Add
ether1 = 192.168.1.2/24 (ISP) ether2 = 192.168.100.1/24 (Server LAN) ether3 = 192.168.200.1/24 (Client LAN) wlan1 = 10.10.1.1/24 (WiFi)

Set Default Route (Jika Ada Internet)

IP → Routes → Add
Dst Address: 0.0.0.0/0 Gateway: 192.168.1.1

Set DNS

IP → DNS
8.8.8.8 8.8.4.4 Centang Allow Remote Requests

Buat DHCP Server

IP → DHCP Server → Setup
ether2 → 192.168.100.10-100 ether3 → 192.168.200.10-100 wlan1 → 10.10.1.10-100

Aktifkan WiFi Access Point

Wireless → wlan1
Mode: ap bridge SSID: LAB-ARULL Security: WPA2 Password: 12345678

Konfigurasi NAT (Jika Ada Internet)

IP → Firewall → NAT → Add
Chain: srcnat Out Interface: ether1 Action: masquerade

Aktifkan Web Server

IP → Services
Enable www (Port 80) Duplicate → Port 8080

Upload File HTML

Menu Files → Upload
index.html

Test Akses

Dari WiFi:
http://10.10.1.1 http://10.10.1.1:8080
Dari LAN:
http://192.168.100.1 http://192.168.200.1

Penjelasan Routing

Semua jaringan berada di satu router, sehingga routing berjalan otomatis tanpa perlu konfigurasi tambahan.

Troubleshooting

Jika tidak bisa akses:
✔ Cek DHCP running ✔ Cek IP client ✔ Cek Service www aktif ✔ Cek Firewall tidak blokir port 80/8080
